.. _glossary:

Glossary
==========

Here is a definition of some terms used in Natron and it's documentation


.. glossary::

   node graph
   	all of the nodes and their connections defining he overall processing of the images


   node
   nodes
   	A single step in the image processing created in the Node Graph. It is displayed as a brick in the Node Graph window.

   parameter
   parameters
   	value allowing to modify the result of a node. (e.g. more or less blurriness for a blur node)
