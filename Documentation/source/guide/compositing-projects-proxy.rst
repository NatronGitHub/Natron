Proxy Mode
==========

User can enable Proxy Mode in the Viewer. On top of the Viewer panel, there is group of 5 button. Click the number 2 button from the right to turn on Proxy Mode. If it turn red, it's on.
The next button to the right is used to define a simple scale factor by which the images are scaled down whenever the proxy mode is activated.

.. image:: _image/viewer_01

As an alternative to letting Nuke generate proxies on the fly, proxy files can be specified using the Proxy File fields in the Read nodes.
