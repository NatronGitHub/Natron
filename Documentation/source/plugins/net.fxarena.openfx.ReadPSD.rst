ReadPSD node
============

|pluginIcon| 

*This documentation is for version 2.7 of ReadPSD.*

Description
-----------

Read Photoshop/GIMP/Cinepaint (RGB/CMYK/GRAY) image formats with ICC color management.

Inputs
------

+---------+---------------+------------+
| Input   | Description   | Optional   |
+=========+===============+============+
| Sync    | Sync          | Yes        |
+---------+---------------+------------+
